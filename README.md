# HollowTrianglePattern_Java
# About the Project
The Java Pattern Generator is a console-based application that prints a variety of visual patterns using stars and numbers. It explores the relationship between row/column indices and character placement, which is a fundamental skill for understanding data structures and algorithm design.

# Features
**Star Pattern:** Generates a geometric shape using character symbols.

**Row-indexed Pattern:** Outputs patterns based on the current row iteration.

**Column-indexed Pattern:** Outputs patterns based on the current column iteration.

**Modular Design:** Each pattern is contained within its own method for better readability and maintenance.

# Technical Stack
**Language:** Java

**Paradigm:** Object-Oriented Programming

**Tools:** Java Development Kit (JDK)
